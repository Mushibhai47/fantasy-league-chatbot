Both are FREE and take 15-20 minutes total

==================================================

1. Go to: https://github.com/new
2. Name: fantasy-league-chatbot
3. Make it PRIVATE (to protect API keys)

==================================================
STEP 3: DEPLOY FRONTEND TO VERCEL
==================================================

1. Go to: https://vercel.com
2. Click "Sign Up" -> "Continue with GitHub"
3. Click "Add New" -> "Project"
4. Import "fantasy-league-chatbot"
5. Configure:
   - Root Directory: frontend
   - Framework: Next.js
   - Build Command: npm run build

6. ADD ENVIRONMENT VARIABLE:
   Name: NEXT_PUBLIC_API_URL
   Value: https://YOUR-RAILWAY-URL.up.railway.app/api
   
   (Replace YOUR-RAILWAY-URL with the URL from Step 2)

7. Click "Deploy"
8. Wait 2-3 minutes
9. COPY YOUR VERCEL URL
   Example: https://fantasy-league-chatbot.vercel.app

==================================================
STEP 4: UPDATE BACKEND CORS
==================================================

1. Go back to Railway
2. Click on your backend service
3. Go to "Variables"
4. Update FRONTEND_URL to your Vercel URL:
   
   FRONTEND_URL=https://your-vercel-url.vercel.app

5. Save (it will redeploy automatically)

==================================================
TROUBLESHOOTING
==================================================

Backend not working?
- Railway -> Your service -> Deployments -> View Logs
- Check for errors

Frontend not working?
- Vercel -> Your project -> Deployments -> Runtime Logs
- Check NEXT_PUBLIC_API_URL is correct

Projections not showing?
- Check Railway logs for "Error fetching projections"
- Verify RAZZBALL_API_KEY is correct
